DOCUMENT TECHNIQUE LUTRON V1
1. Objectif Général
LUTRON est une plateforme centralisée de supervision, administration et gestion technique basée sur une architecture modulaire.
Le système doit permettre de superviser plusieurs types d'infrastructures :
* Applications Web
* APIs
* Machines physiques
* Machines virtuelles
* Services réseau
* Systèmes de jeux
* Outils internes
* Services tiers
Le Service constitue l'entité centrale de toute la plateforme.
Toutes les ressources, permissions, modules, historiques et données sont rattachés à un Service.

* Gestion de parc
* Gestion de déploiement
* Gestion de logs
* Gestion CI/CD
* Gestion des incidents
* Gestion des certificats SSL
* Gestion Kubernetes

sans modification du Core.
